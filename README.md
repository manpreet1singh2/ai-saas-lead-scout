# PineLine Plywood

Initial skeleton for a plywood e-commerce website.

## Stack
- Next.js / React frontend
- Node.js route handlers
- PostgreSQL-ready data layer
- n8n webhook support for quote automation

## Pages
- Home
- Catalog
- Admin dashboard

## API
- GET/POST `/api/products`
- PATCH/DELETE `/api/products/:id`
- POST `/api/quotes`

## Environment
Copy `.env.example` to `.env.local`.
